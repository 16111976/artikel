# Eselsbrücken (kuratiert)

| Wort | Eselsbruecke |
|---|---|
| Zeitung | Viele Endungen auf -ung sind fast immer feminin: die Zeitung. |
| Freiheit | -heit zeigt oft auf `die` hin: die Freiheit. |
| Brötchen | Diminutive mit -chen sind immer sächlich: das Brötchen. |
| Mädchen | Trotz weiblicher Bedeutung bleibt es wegen -chen: das Mädchen. |
| Freundschaft | -schaft ist meistens feminin: die Freundschaft. |
| Computer | Fremdwörter auf -er sind oft maskulin: der Computer. |
